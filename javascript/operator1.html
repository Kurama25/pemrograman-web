Operator adalah simbol khusus yang digunakan untuk melakukan operasi pada nilai dan variabel. JavaScript memiliki berbagai jenis operator seperti aritmatika untuk perhitungan matematika, penugasan untuk menyimpan nilai, dan perbandingan untuk membandingkan nilai.

Operator Aritmatika

üí° Apa itu Operator Aritmatika?

Operator aritmatika digunakan untuk melakukan operasi matematika pada angka (number). Operator ini memungkinkan kita untuk melakukan perhitungan seperti penjumlahan, pengurangan, perkalian, pembagian, dan operasi matematika lainnya.

‚ûï
Penjumlahan
+

Menjumlahkan dua nilai atau lebih. Dapat juga digunakan untuk menggabungkan string.

let a = 10; let b = 5; let hasil = a + b; // 15 // Untuk string let nama = "Budi" + " Santoso"; // "Budi Santoso"
‚ûñ
Pengurangan
-

Mengurangkan satu nilai dari nilai lainnya.

let a = 20; let b = 8; let hasil = a - b; // 12 // Contoh lain let saldo = 100000 - 25000; // 75000
‚úñÔ∏è
Perkalian
*

Mengalikan dua nilai atau lebih.

let a = 6; let b = 4; let hasil = a * b; // 24 // Menghitung luas let luas = 5 * 8; // 40
‚ûó
Pembagian
/

Membagi satu nilai dengan nilai lainnya.

let a = 20; let b = 4; let hasil = a / b; // 5 // Membagi dengan desimal let rata = 75 / 2; // 37.5
üì¶
Modulus (Sisa Bagi)
%

Mengembalikan sisa hasil pembagian (remainder).

let a = 17; let b = 5; let sisa = a % b; // 2 // Cek bilangan genap/ganjil let cek = 10 % 2; // 0 (genap)
‚¨ÜÔ∏è
Eksponen (Pangkat)
**

Memangkatkan nilai pertama dengan nilai kedua.

let a = 2; let b = 3; let hasil = a ** b; // 8 (2¬≥) // Contoh lain let kuadrat = 5 ** 2; // 25
‚ûï
Increment
++

Menambah nilai variabel sebesar 1.

let angka = 5; angka++; // angka sekarang 6 // Sama dengan: angka = angka + 1;
‚ûñ
Decrement
--

Mengurangi nilai variabel sebesar 1.

let angka = 10; angka--; // angka sekarang 9 // Sama dengan: angka = angka - 1;
Operator Nama Contoh Hasil
+ Penjumlahan 10 + 5 15
- Pengurangan 10 - 5 5
* Perkalian 10 * 5 50
/ Pembagian 10 / 5 2
% Modulus 10 % 3 1
** Eksponen 2 ** 3 8
++ Increment x++ x + 1
-- Decrement x-- x - 1

Operator Penugasan

üí° Apa itu Operator Penugasan?

Operator penugasan (assignment) digunakan untuk memberikan atau menyimpan nilai ke dalam variabel. Operator dasar adalah = (sama dengan), dan ada juga operator penugasan kombinasi yang menggabungkan operasi aritmatika dengan penugasan.

üìå
Penugasan Dasar
=

Memberikan nilai ke variabel.

let nama = "Budi"; let umur = 25; let aktif = true; // Mengubah nilai umur = 26;
‚ûï
Penugasan Penjumlahan
+=

Menambah nilai variabel dengan nilai tertentu.

let skor = 10; skor += 5; // skor = 15 // Sama dengan: skor = skor + 5;
‚ûñ
Penugasan Pengurangan
-=

Mengurangi nilai variabel dengan nilai tertentu.

let stok = 100; stok -= 25; // stok = 75 // Sama dengan: stok = stok - 25;
‚úñÔ∏è
Penugasan Perkalian
*=

Mengalikan nilai variabel dengan nilai tertentu.

let harga = 1000; harga *= 2; // harga = 2000 // Sama dengan: harga = harga * 2;
‚ûó
Penugasan Pembagian
/=

Membagi nilai variabel dengan nilai tertentu.

let total = 100; total /= 4; // total = 25 // Sama dengan: total = total / 4;
üì¶
Penugasan Modulus
%=

Menyimpan sisa pembagian ke variabel.

let nilai = 17; nilai %= 5; // nilai = 2 // Sama dengan: nilai = nilai % 5;
Operator Nama Contoh Sama Dengan
= Penugasan x = 5 x = 5
+= Tambah & Tugaskan x += 3 x = x + 3
-= Kurang & Tugaskan x -= 3 x = x - 3
*= Kali & Tugaskan x *= 3 x = x * 3
/= Bagi & Tugaskan x /= 3 x = x / 3
%= Modulus & Tugaskan x %= 3 x = x % 3

Operator Perbandingan

üí° Apa itu Operator Perbandingan?

Operator perbandingan (comparison) digunakan untuk membandingkan dua nilai dan mengembalikan nilai boolean (true atau false). Operator ini sangat penting untuk membuat keputusan dalam program.

üü∞
Sama Dengan (Longgar)
==

Memeriksa apakah dua nilai sama (tanpa memeriksa tipe data).

5 == 5 // true 5 == "5" // true (tipe berbeda) 10 == 5 // false let umur = 17; umur == 17 // true
‚úÖ
Sama Dengan (Ketat)
===

Memeriksa apakah nilai dan tipe data sama (lebih disarankan).

5 === 5 // true 5 === "5" // false (tipe berbeda) 10 === 5 // false let nama = "Budi"; nama === "Budi" // true
‚ùå
Tidak Sama (Longgar)
!=

Memeriksa apakah dua nilai tidak sama (tanpa tipe data).

5 != 10 // true 5 != "5" // false 5 != 5 // false let status = "aktif"; status != "nonaktif" // true
üö´
Tidak Sama (Ketat)
!==

Memeriksa apakah nilai atau tipe data tidak sama (lebih disarankan).

5 !== 10 // true 5 !== "5" // true (tipe berbeda) 5 !== 5 // false let angka = 100; angka !== "100" // true
‚¨ÜÔ∏è
Lebih Besar
>

Memeriksa apakah nilai pertama lebih besar dari nilai kedua.

10 > 5 // true 5 > 10 // false 5 > 5 // false let nilai = 85; nilai > 75 // true
‚¨áÔ∏è
Lebih Kecil
<

Memeriksa apakah nilai pertama lebih kecil dari nilai kedua.

5 < 10 // true 10 < 5 // false 5 < 5 // false let umur = 15; umur < 17 // true
‚¨ÜÔ∏è‚úÖ
Lebih Besar Sama Dengan
>=

Memeriksa apakah nilai pertama lebih besar atau sama dengan nilai kedua.

10 >= 5 // true 10 >= 10 // true 5 >= 10 // false let skor = 70; skor >= 70 // true (lulus)
‚¨áÔ∏è‚úÖ
Lebih Kecil Sama Dengan
<=

Memeriksa apakah nilai pertama lebih kecil atau sama dengan nilai kedua.

5 <= 10 // true 10 <= 10 // true 15 <= 10 // false let stok = 5; stok <= 10 // true (stok rendah)
Operator Nama Contoh Hasil
== Sama dengan (longgar) 5 == "5" true
=== Sama dengan (ketat) 5 === "5" false
!= Tidak sama (longgar) 5 != "5" false
!== Tidak sama (ketat) 5 !== "5" true
> Lebih besar 10 > 5 true
< Lebih kecil 5 < 10 true
>= Lebih besar sama dengan 10 >= 10 true
<= Lebih kecil sama dengan 5 <= 10 true
‚ö†Ô∏è Perbedaan == dan ===

Operator == (loose equality) membandingkan nilai tanpa memeriksa tipe data. JavaScript akan melakukan konversi tipe data jika perlu.

Operator === (strict equality) membandingkan nilai DAN tipe data. Lebih disarankan karena lebih aman dan menghindari bug.

Contoh: 5 == "5" hasilnya true, tetapi 5 === "5" hasilnya false karena tipe datanya berbeda (number vs string).
