TARGET

Siswa dianggap berhasil jika program yang dibuat dapat menampilkan output seperti contoh di bawah ini. Angka boleh berbeda sesuai dengan input yang kalian masukkan, tetapi format dan struktur output harus sama persis.

üìã Kriteria Keberhasilan:

  3. Rumus perhitungan totalKotor, totalBersih, pajak, dan totalAkhir sudah tepat
  4. Output ditampilkan menggunakan console.log() dengan format yang rapi
  5. Berhasil menjelaskan error pada bagian debugging
üí° Tips Menampilkan Output:

‚Ä¢ Gunakan console.log() untuk menampilkan setiap baris
‚Ä¢ Gunakan tanda + untuk menggabungkan teks dan variabel
‚Ä¢ Contoh: console.log("Total Kotor: Rp. " + totalKotor)
‚Ä¢ Tambahkan console.log("===...===") untuk garis pembatas
