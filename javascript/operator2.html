Operator Logika

üí° Apa itu Operator Logika?

Operator logika digunakan untuk menggabungkan atau membalikkan kondisi boolean (true/false). Operator ini sangat berguna dalam membuat keputusan yang lebih kompleks dalam program, seperti memeriksa apakah beberapa kondisi terpenuhi secara bersamaan atau salah satunya terpenuhi.

üîó
AND (DAN)
&&

Mengembalikan true hanya jika SEMUA kondisi bernilai true. Jika salah satu kondisi false, hasilnya false.

let umur = 18; let punyaKTP = true; // Cek apakah bisa buat rekening let bisaBuatRekening = umur >= 17 && punyaKTP; // true (kedua kondisi terpenuhi) // Contoh lain let hasil = true && true; // true hasil = true && false; // false hasil = false && true; // false
üîÄ
OR (ATAU)
||

Mengembalikan true jika SALAH SATU atau LEBIH kondisi bernilai true. Hanya false jika semua kondisi false.

let adalDiskon = true; let adalVoucher = false; // Cek apakah dapat potongan harga let dapatPotongan = adalDiskon || adalVoucher; // true (salah satu kondisi terpenuhi) // Contoh lain let hasil = true || false; // true hasil = false || false; // false hasil = true || true; // true
üîÑ
NOT (TIDAK)
!

Membalikkan nilai boolean. True menjadi false, dan false menjadi true.

let sudahLogin = false; let belumLogin = !sudahLogin; // true (kebalikan dari false) // Contoh lain let aktif = true; let nonAktif = !aktif; // false // Double NOT untuk konversi ke boolean let nilai = "teks"; let isTruthy = !!nilai; // true
Operator Nama Contoh Hasil Keterangan
&& AND true && true true Semua harus true
&& AND true && false false Salah satu false
|| OR true || false true Salah satu true
|| OR false || false false Semua false
! NOT !true false Membalikkan nilai
! NOT !false true Membalikkan nilai
‚úÖ Tips Praktis

Operator AND (&&): Gunakan ketika semua kondisi harus dipenuhi. Contoh: validasi form (nama terisi DAN email terisi).

Operator OR (||): Gunakan ketika salah satu kondisi cukup. Contoh: metode pembayaran (cash ATAU transfer ATAU kartu kredit).

Operator NOT (!): Gunakan untuk membalikkan kondisi. Contoh: !sudahLogin untuk mengecek apakah pengguna belum login.

Operator String

üí° Apa itu Operator String?

Operator string digunakan untuk memanipulasi dan menggabungkan teks (string). Operator utama adalah tanda plus (+) untuk menggabungkan (concatenation) string. JavaScript juga mendukung template literals dengan backtick (`) untuk cara yang lebih modern dan mudah dalam menggabungkan string dengan variabel.

‚ûï
Penggabungan String
+

Menggabungkan dua atau lebih string menjadi satu string yang lebih panjang. Metode klasik yang masih banyak digunakan.

let namaDepan = "Budi"; let namaBelakang = "Santoso"; let namaLengkap = namaDepan + " " + namaBelakang; // "Budi Santoso" // Menggabungkan string dengan angka let umur = 25; let kalimat = "Saya berumur " + umur + " tahun"; // "Saya berumur 25 tahun"
üìã
Template Literals
`${}`

Cara modern untuk menggabungkan string dengan variabel menggunakan backtick (`) dan ${variabel}. Lebih mudah dibaca dan fleksibel.

let nama = "Siti"; let umur = 22; let kota = "Jakarta"; // Menggunakan template literals let profil = `Nama: ${nama}, Umur: ${umur}, Kota: ${kota}`; // "Nama: Siti, Umur: 22, Kota: Jakarta" // Multi-line string let pesan = `Halo ${nama}! Selamat datang di sistem kami. Anda terdaftar dari ${kota}.`;
üîó
Penugasan Penggabungan
+=

Menambahkan string ke variabel yang sudah ada. Shorthand untuk menggabungkan dan menyimpan hasil ke variabel yang sama.

let pesan = "Halo"; pesan += " dunia"; // pesan sekarang "Halo dunia" pesan += "!"; // pesan sekarang "Halo dunia!" // Sama dengan: pesan = pesan + " JavaScript";
üî¢
Ekspresi dalam Template
${expr}

Anda bisa menyisipkan ekspresi JavaScript apa pun di dalam ${}, termasuk operasi matematika, pemanggilan fungsi, dan kondisi.

let harga = 50000; let jumlah = 3; // Operasi matematika di dalam ${} let total = `Total: Rp ${harga * jumlah}`; // "Total: Rp 150000" // Kondisi di dalam ${} let skor = 85; let status = `Nilai: ${skor >= 75 ? 'Lulus' : 'Tidak Lulus'}`; // "Nilai: Lulus"
‚ö†Ô∏è Perhatian: Perbedaan + dan Template Literals

Ketika menggunakan operator +, jika salah satu operand adalah string, JavaScript akan mengubah yang lain menjadi string juga.

Contoh: "5" + 3 menghasilkan "53" (string), bukan 8 (number).
Contoh: 5 + 3 + " apel" menghasilkan "8 apel" karena 5 dan 3 dijumlahkan dulu.
Contoh: "Ada " + 5 + 3 menghasilkan "Ada 53" karena 5 diubah ke string terlebih dahulu.

Template literals selalu menghasilkan string, lebih konsisten dan aman.

Metode Contoh Hasil
Concatenation (+) "Halo" + " " + "Dunia" "Halo Dunia"
Template Literals `Halo ${"Dunia"}` "Halo Dunia"
String + Number "Umur: " + 25 "Umur: 25"
Template + Expr `Total: ${10 * 5}` "Total: 50"
Assignment (+=) str = "A"; str += "B" "AB"

Operator Typeof

üí° Apa itu Operator Typeof?

Operator typeof digunakan untuk memeriksa tipe data dari sebuah variabel atau nilai. Operator ini mengembalikan string yang menunjukkan tipe data, seperti "string", "number", "boolean", "object", "function", atau "undefined". Sangat berguna untuk validasi dan debugging.

üî§
Typeof untuk String
typeof

Memeriksa apakah suatu nilai bertipe string (teks).

let nama = "Budi"; let tipe = typeof nama; console.log(tipe); // "string" // Dengan tanda kurung console.log(typeof("Hello")); // "string" // Validasi input if (typeof nama === "string") { console.log("Nama valid!"); }
üî¢
Typeof untuk Number
typeof

Memeriksa apakah suatu nilai bertipe number (angka).

let umur = 25; console.log(typeof umur); // "number" let harga = 99.99; console.log(typeof harga); // "number" // NaN juga bertipe "number" console.log(typeof NaN); // "number"
‚úÖ
Typeof untuk Boolean
typeof

Memeriksa apakah suatu nilai bertipe boolean (true/false).

let aktif = true; console.log(typeof aktif); // "boolean" let selesai = false; console.log(typeof selesai); // "boolean" // Hasil perbandingan console.log(typeof (5 > 3)); // "boolean"
üì¶
Typeof untuk Object & Array
typeof

Memeriksa object dan array. Perhatian: array juga mengembalikan "object".

let orang = { nama: "Budi" }; console.log(typeof orang); // "object" let angka = [1, 2, 3]; console.log(typeof angka); // "object" ‚ö†Ô∏è // null juga "object" (bug historis) console.log(typeof null); // "object" ‚ö†Ô∏è // Cek array dengan Array.isArray() console.log(Array.isArray(angka)); // true
‚ö°
Typeof untuk Function
typeof

Memeriksa apakah suatu nilai adalah function (fungsi).

function sapa() { return "Halo!"; } console.log(typeof sapa); // "function" // Arrow function let tambah = (a, b) => a + b; console.log(typeof tambah); // "function"
‚ùì
Typeof untuk Undefined
typeof

Memeriksa apakah variabel belum diberi nilai (undefined).

let x; console.log(typeof x); // "undefined" // Variabel yang tidak dideklarasikan console.log(typeof tidakAda); // "undefined" // Cek apakah variabel sudah didefinisikan if (typeof myVar !== "undefined") { // Aman untuk menggunakan myVar }
Nilai typeof Hasil Keterangan
"Halo" "string" Tipe string
123 "number" Tipe number
true "boolean" Tipe boolean
{} "object" Tipe object
[] "object" Array adalah object
null "object" Bug historis JS
function(){} "function" Tipe function
undefined "undefined" Belum diberi nilai
‚ö†Ô∏è Quirks typeof yang Perlu Diperhatikan

1. Array mengembalikan "object": Untuk mengecek array, gunakan Array.isArray() bukan typeof.

2. null mengembalikan "object": Ini bug historis JavaScript yang tidak diperbaiki untuk backward compatibility. Gunakan nilai === null untuk mengecek null.

3. NaN mengembalikan "number": Meskipun artinya "Not a Number", NaN bertipe number. Gunakan isNaN() atau Number.isNaN() untuk mengecek NaN.

Operator Kondisional (Ternary)

üí° Apa itu Operator Ternary?

Operator ternary adalah operator kondisional yang menyediakan cara singkat untuk menulis kondisi if-else dalam satu baris. Sintaksnya: kondisi ? nilaiJikaTrue : nilaiJikaFalse. Operator ini sangat berguna untuk membuat kode lebih ringkas dan mudah dibaca, terutama untuk kondisi sederhana.

üéØ
Ternary Sederhana
? :

Penggunaan dasar operator ternary untuk menggantikan if-else sederhana.

let umur = 18; // Menggunakan ternary let status = umur >= 17 ? "Dewasa" : "Anak-anak"; // "Dewasa" // Sama dengan if-else: let status2; if (umur >= 17) { status2 = "Dewasa"; } else { status2 = "Anak-anak"; }
üî¢
Ternary dengan Angka
? :

Menggunakan ternary untuk mengembalikan nilai numerik berdasarkan kondisi.

let skor = 85; // Menentukan bonus let bonus = skor >= 80 ? 1000000 : 500000; // 1000000 // Diskon produk let totalBelanja = 300000; let diskon = totalBelanja > 250000 ? 0.1 : 0; let bayar = totalBelanja - (totalBelanja * diskon);
üîó
Ternary Bersarang
? : ? :

Operator ternary dapat disarangkan untuk menangani multiple kondisi (gunakan dengan hati-hati agar tidak terlalu kompleks).

let nilai = 75; // Menentukan grade let grade = nilai >= 90 ? "A" : nilai >= 80 ? "B" : nilai >= 70 ? "C" : nilai >= 60 ? "D" : "E"; // "C" // Lebih baik gunakan if-else jika terlalu kompleks
üìù
Ternary dalam Template
? :

Menggunakan operator ternary di dalam template literals untuk output dinamis.

let sudahLogin = true; let username = "Budi"; // Pesan sambutan let pesan = `Halo, ${sudahLogin ? username : 'Tamu'}!`; // "Halo, Budi!" // Status member let poin = 150; let status = `Status: ${poin > 100 ? 'Gold' : 'Silver'}`;
‚ö°
Ternary dalam Fungsi
? :

Menggunakan operator ternary sebagai return value dalam fungsi.

// Fungsi cek ganjil/genap function cekGanjilGenap(angka) { return angka % 2 === 0 ? "Genap" : "Ganjil"; } console.log(cekGanjilGenap(4)); // "Genap" // Arrow function dengan ternary const maksimal = (a, b) => a > b ? a : b; console.log(maksimal(10, 5)); // 10
üé®
Ternary untuk Class/Style
? :

Menggunakan ternary untuk menentukan class CSS atau style secara dinamis (berguna dalam manipulasi DOM).

let isAktif = true; // Menentukan class CSS let className = isAktif ? "active" : "inactive"; // Menentukan warna let skor = 85; let warna = skor >= 75 ? "green" : "red"; // Dalam HTML (contoh) // element.className = isAktif ? "active" : "inactive";
If-Else Ternary Equivalent
if (x > 5) { y = 10; } else { y = 20; } y = x > 5 ? 10 : 20;
if (login) { msg = "Hi"; } else { msg = "Bye"; } msg = login ? "Hi" : "Bye";
if (age >= 18) { return true; } else { return false; } return age >= 18 ? true : false;
‚úÖ Kapan Menggunakan Ternary?

GUNAKAN ternary ketika:
‚Ä¢ Kondisi sederhana dengan dua pilihan jelas
‚Ä¢ Untuk assignment atau return value langsung
‚Ä¢ Membuat kode lebih ringkas dan mudah dibaca
‚Ä¢ Di dalam template literals atau JSX (React)

JANGAN gunakan ternary ketika:
‚Ä¢ Kondisi sangat kompleks atau bersarang dalam
‚Ä¢ Ada lebih dari 2-3 kondisi
‚Ä¢ Perlu menjalankan multiple statements
‚Ä¢ Membuat kode susah dibaca (gunakan if-else biasa)
